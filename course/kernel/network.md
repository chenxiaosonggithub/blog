# 套接字

套接字接口（socket interface）是两台机器的进程间进行通信的一种方法。这里顺便提一下，Windows系统通过Windows Sockets技术规范（简称WinSock）实现套接字接口，套接字服务由`Winsock.dll`提供。

套接字（socket）是一种通信机制，让两台机器的开发工作可以跨网络进行。
